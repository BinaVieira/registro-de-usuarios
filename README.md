# Sistema de Gerenciamento de Usuários

Este é um projeto simples de **gerenciamento de usuários** desenvolvido com **Node.js** no backend e **HTML/CSS/JavaScript** no frontend. Ele permite listar e cadastrar usuários através de uma interface amigável.

## Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (fetch API)
- **Backend:** Node.js, Express, CORS
- **Arquitetura:** MVC (Model-View-Controller) simplificada
- **Banco de dados:** Simulado em memória (array de objetos)

## Funcionalidades

- **Listar usuários:** Exibe todos os usuários cadastrados.
- **Cadastrar usuário:** Adiciona um novo usuário (nome, e-mail, idade) via formulário.
